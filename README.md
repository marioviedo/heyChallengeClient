# heyclient

## 1.- Project setup
```
npm install
```

### 2- Install Sass compiler 
```
npm install -g sass
```

### 3- Runs Sass watcher
```
sass --watch src/sass:public
```

### Compiles and hot-reloads for development
```
npm run serve
```


### Compiles and minifies for production
```
npm run build
```
